# aws-frame-coloring
- Google ChromeでAWSのマネジメントコンソールのヘッダーとフッダーをアカウント名で変更する
- ex) stg/dev/prd

### 使い方
- git clone git@github.com
- Google ChromeのURLで `chrome://extensions` を開く
- `デベロッパーモード` にチェック
- `パッケージ化されていない拡張機能` を読み込む
- cloneしたディレクトリを選択
- 詳細を選択し、`シークレットモードでの実行を許可する`を有効にする
